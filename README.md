# CRUD Operations using Flask ORM

A simple CRUD (Create, Read, Update, Delete) web application built using Flask and SQLAlchemy ORM.

## Project Output

![Application Preview](outputs/app_preview.png)

## Features

* Add records
* View records
* Update records
* Delete records
* SQLite database support
* Simple dashboard UI

## Technologies Used

* Python
* Flask
* SQLAlchemy ORM
* SQLite
* HTML
* CSS

## Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
